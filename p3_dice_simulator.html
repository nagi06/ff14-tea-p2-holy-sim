デバフ抽選：番号1×3人／2×2人／3×3人をランダム付与。ジャンプ種別は全体でハイジャンプ×4・スパイン(↑)×2・イルーシヴ(↓)×2。2番の2人は必ず矢印(スパインorイルーシヴ)持ち。残り2つの矢印は1番/3番のメンバーにランダム。
ジャンプ種別の意味：ハイジャンプ＝自分の足元に塔。スパイン(↑)＝前方に塔→ボスを向いて立つ。イルーシヴ(↓)＝後方に塔→ボスに背を向けて立つ。
散開優先度(PT取り決め)：西 H1/MT/ST/D1/D2/D3/D4/H2 東(H1が最西優先、H2が最東優先)。同じ役目が複数人いる時はこの順で西から埋める。矢印固定：イルーシヴ→西(D)、スパイン→東(B)。矢印持ちの位置を先に固定し、残りを優先度で埋める。
塔の出現(簡略・固定)：1番塔3本＝西(D)・南(C)・東(B)。2番塔2本＝北西寄り・北東寄り。3番塔3本＝西・南・東(1番と同位置)。
連旋：「内→外」か「外→内」を開始時に抽選。1回目・2回目でその通り／逆に内外をタップで回答。
時系列：①初期散開(1番=西/南/東、他=北に集合) → ②1番着弾(塔3本・連旋1回目) → ③塔踏み1(3番が西/南/東、2番は北西/北東へ散開) → ④2番着弾(塔2本・連旋2回目) → ⑤塔踏み2(初期散開で西/東にいた1番の2人が北西/北東塔、3番は西/南/東へ散開) → ⑥3番着弾(塔3本) → ⑦塔踏み3(2番2人＋南にいた1番1人が優先度順に西/南/東) → ⑧最終配置確認 → ⑨4塔散開(斜め4方向にペアで散開) → ⑩4塔踏み(必要人数抽選・調整役が優先度で調整) → ⑪ソウルテザー(MT=分身線/ST=ボス線/他は南待機) → ⑫リザルト。
簡略化した点：塔の詳細な出現角度は固定モデル。ゲイルスコグル(直線範囲)は省略。2番の立ち位置は北西/北東の2択に簡略化。連旋は中央寄り/外周寄りの2択タップで判定。
4塔(ダークドラゴンダイブ)：斜め4方向(北東/南東/南西/北西)にペアで散開。固定役(H1/H2/D3/D4)は動かず、調整役(MT/ST/D1/D2)がMT>ST>D1>D2の優先度で 自枠>左>右>対角 の順に空きのある塔へ調整。塔の柱の数=必要人数(1〜4・合計8)。
ソウルテザー：MT=分身線(無敵受け)、ST=ボス線(激減受け)、他6人は南に集合して待機。
実際の座標・タイミングにはランダム要素や細かい前後位置があるため、本ツールは練習用に簡略化したモデルです。必ず自分のPTの取り決めと照らし合わせてください。
